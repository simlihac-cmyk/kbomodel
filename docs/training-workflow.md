# Training Workflow (Deprecated)

학습 기반 승률 보정 실험 워크플로는 현재 운영 경로에서 사용하지 않습니다.

현재 운영 모델 원칙:
- 승률은 전년도 기준점, 현재 승률, 최근 10경기, 홈/원정 성향, 기대 득점 기반의 규칙 모델로 계산합니다.
- 별도의 학습 기반 direct/contextual 확률 보정은 운영에 연결하지 않습니다.
- `패스 / 관심 / 픽 / 강한 픽` 레이어만 Elo, 최근 흐름, 휴식일 같은 사실 기반 신호로 계산합니다.

즉, 노트북 학습 결과를 만들어 `promote`하는 흐름은 더 이상 공식 운영 절차가 아닙니다.
